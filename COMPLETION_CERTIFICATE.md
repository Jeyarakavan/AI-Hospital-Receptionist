## 💻 TECHNICAL SPECIFICATIONS

### Frontend
- **Framework:** React 18.2 with Vite 5.4
- **Styling:** Tailwind CSS 3.4 + 150+ custom utilities
- **Icons:** Lucide React (professional medical icons)
- **Routing:** React Router v6 with protected routes
- **State Management:** Context API + hooks
- **Real-time:** Socket.IO client integration
- **Notifications:** React Hot Toast
- **Forms:** React Hook Form

### Backend
- **Framework:** Express.js
- **Real-time:** Socket.IO
- **Query Builder:** Knex.js
- **Database:** SQLite (dev) / PostgreSQL (prod)
- **Security:** SHA-256 password hashing
- **API Endpoints:** 8+ staff management endpoints

### Database
- **Tables:** staff, staff_credentials, appointments, calls
- **Schema:** 21+ columns per table
- **Relationships:** Foreign keys, constraints
- **Migrations:** Auto-create on startup

### Design
- **Color Palette:** Teal, Dark Blue, Health Green, Alert Red
- **Animations:** Fade, slide, float, pulse effects
- **Responsive:** Mobile-first, all screen sizes
- **Accessibility:** Semantic HTML, ARIA labels
- **Performance:** Optimized images, lazy loading
